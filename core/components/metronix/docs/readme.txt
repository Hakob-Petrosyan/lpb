        MetroniX (MODX Manager Theme)

Since: October 16 2016
Author: Adorazel
License: GNU GPLv2 (or later at your option)

WHAT IT DOES
---------------
This theme just overwrite the default MODX CSS styles with a metronix.min.css.
When uninstalling, the default theme is back.

REQUIREMENT
---------------
Tested on MODx Revolution 2.5+ and 2.6+

NOTES
---------------
If you're not activating the theme via the installer,
don't forget to change the "manager_theme" setting to "metronix".
